Youtube MongoDB Scraper

Youtube scraper is a youtube homepage web scraper that uses axios, cheerio, and mongoose to scrape, sort and store video data. The webpage is deployed via heroku and utilizes the mLab MongoDB addon to store data. Server framework uses express and express-handlebars to load large amounts of data. The webpage also allows for the creation and deletion of public playlists which are stored in the database. All user interaction is handled from the top navbar or from links below each video to keep things simple!
